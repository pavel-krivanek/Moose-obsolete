I am an attach point that will end the line on the border of the end shapes.

Unline ContinuousAttachPoint & co. you don't need to care about what the end shape is, as long as it is one of the basic shapes.

Supported shapes:
	- Box
	-RoundedBox
	- Ellipse/Circle

Supported lines:
	- Line (straight line)
